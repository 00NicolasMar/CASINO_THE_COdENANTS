# SpartanRoulette
Este juego es una ruleta que consta de 37 casillas, se√±aladas con n√∫meros del 0 al 36, de los cuales 18 son n√∫meros identificados por color negro, los 18 restantes son de color rojo y el 0 siendo el √∫nico n√∫mero color verde de toda la ruleta. Este juego cuenta con la opci√≥n de ingresar un monto de dinero, y seleccionar las siguientes opciones de apuesta, la cual generar√° una ganancia correspondiente a lo seleccionado.
### Opciones de apuesta:
#### Escoger color
 - Los colores rojo y negro generan una ganancia de **X2 sobre el dinero apostado**.
 - El color Verde genera una ganancia del **X50** sobre el dinero apostado, siendo este el premio mayor de la ruleta. 
#### N√∫mero 
 - La opci√≥n de escoger un n√∫mero est√°, y siempre y cuando se acierte el n√∫mero, la ganancia obtenida sera de **X5** sobre el valor apostado.
#### Par o Impar
 - Si se acierta que es par o impar, la ganancia ser√° de **X1.5 veces el valor apostado**.
El hecho de que existan estas opciones de apuesta no implica que sea necesario seleccionarlas todas, por lo que si no desea ingresar en su apuesta algunas de las anteriores opciones, puede expresar ‚Äúninguno‚Äù, lo que indicar√° que no se apostara en tal apartado. (debe seleccionar al menos 1 de las opciones de apuesta, o si no no se har√° efectiva la apuesta).
#### Si no acerta alguna de las opciones seleccionadas, el jugador pierde el dinero apostado.
